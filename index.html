How to Creat Your Own Web Page In GitHub

The following are simple steps to follow to create a simple personal GitHub page.

1- open Github web page and Sign up to create new account if you don't have any, otherwise sign in.

2-Follow the prompts as you fill the blanks to set up your account credentials and settings.


3- Once you set an account and your signed in to your account,from your github homepage go to "Repositeries" tab to create new repository where the personal webpage files reside


Note: you might be able to start a new repository from this homepage also once you are signed in.

4- Name your repository, and select "Public" which will allow anyone on the internet to see your webpage. you can keep the rest in default or customize it to your needs.

5- Once you repository is created you will be able to add your web project files either by upload otion or create files in the repository.


6- Once you added the files in the repository, the next step is to click Commit changes so the files will be moved from staging to final in the repository and will be used to display the webpage.


7- Once the files are committed , you should be able to check them under "Code" tab as they are committed for confirmation.

8- last thing is to check and test your github files pages
9- Under "Settings" tab, go to "Pages" on the right list column 10- under this page you can visit you site or copy past the repository address and parse it to new webpage to verify.
